## 时间/空间复杂度

### 时间复杂度是什么？
- 一个函数，用大O标识，比如 O(1)、O(n)、O(logN)...
- 定性描述该算法的运行时间
- O(1) < O(logN) < O(n) < O(nlogn) < O(n^2)

### 空间复杂度是什么？
- 一个函数，用大O标识，比如 O(1)、O(n)、O(logN)...
- 算法在运行过程中临时占用存储空间大小的量度
- O(1) < O(logN) < O(n) < O(nlogn) < O(n^2)
